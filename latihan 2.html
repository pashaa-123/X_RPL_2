Perjalanan Mengenal HTML

Dulu,saya tidak tahu apa itu HTML. Saya hanya melihat halaman web dan mengira itu dibuat dengan cara rumit. Namun setelah saya bealajar, saya menyadari bahwa HTML adalah dasar dari setiap halaman web. HTML, atau Hyper Text Markup Language, digunakan untuk membangun struktur dari sebuah halaman.
